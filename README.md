# ScreenTime

A modern movie and TV discovery application built with React Native and TypeScript.

ScreenTime uses data from The Movie Database (TMDB) API to provide users with an intuitive and visually appealing way to browse trending movies, TV shows, and detailed entertainment information. This project was created as a hands-on learning experience to strengthen my skills in React Native, React, TypeScript, mobile UI development, and API integration.

## Features

* Browse trending and popular movies
* Explore TV shows and entertainment content
* View detailed movie information
* View detailed TV show information
* Modern and responsive mobile user interface
* Smooth navigation between screens
* Real-time data powered by TMDB

## Tech Stack

This project is built with the following technologies:

* React Native 0.68
* React 17
* TypeScript
* React Navigation v6
* React Native Vector Icons
* React Native Linear Gradient
* Axios
* The Movie Database (TMDB) API

## Project Goals

The primary goal of this project is to explore and implement modern React Native development practices, including:

* Component-driven architecture
* API consumption and data management
* Mobile navigation patterns
* Reusable UI components
* Type-safe development with TypeScript
* Responsive and polished mobile interfaces
